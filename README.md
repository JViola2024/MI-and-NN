#Artificial Bee Colony (ABC)

Ez a program a méhraj optimalizáció működését mutatja be, C# nyelven.  
A célfüggvény, amit a méhek optimalizálnak a következő--> f(x, y)=x²+y². A program ennek a minimumát próbálja megtalálni.
A keresés során háromféle méh vesz részt:
-Dolgozó méhek–saját megoldásaikat próbálják javítani.
-Megfigyelő méhek–a sikeres méhek körül keresnek jobb pozíciókat.
-Felderítő méhek–ha valaki régóta nem talál jobb megoldást, új helyről indul.
A program minden néhány iteráció után kiírja, hol tart a keresés, és végül megadja a legjobb talált megoldást.

#paraméterek
- Méhek száma: 20  
- Iterációk száma: 50  
- Keresési tartomány: -5 és +5 között
